Detailed Description

Abstract builder interface for composing Plant objects.

PlantBuilder defines the interface for constructing composite Plant instances (for example, groups wrapped in a decorative pot or gift wrap). Concrete builders should implement buildCompositePlant to return a new Plant pointer representing the composed product.
