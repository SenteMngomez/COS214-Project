Detailed Description

A Section manages a collection of Person instances, maintains a history of messages, and provides an interface to notify participants when a Person sends a message or makes a request. Concrete Section types implement the notify(Person*) method to define how messages are propagated Sections also provide an iterator for traversing the contained chat history.
