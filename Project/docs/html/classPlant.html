Detailed Description

Abstract base class representing a plant in the nursery.

Plant encapsulates the core behaviour of nursery plants, including state management (via PlantState), care strategies (via CareStrategy), and lifecycle operations. Supports Prototype pattern through cloning and Decorator pattern through wrapping.
