Detailed Description

Command used to sell one or more plants and generate a receipt.

The SellPlant command removes plants from the greenhouse inventory (via Greenhouse interfaces), constructs a composite plant when needed, and uses a ReceiptBuilder/SalesAssistance to produce a Receipt object.
