Detailed Description

A HelpDesk is a concrete Section that broadcasts messages to all participants and routes help requests to the admin or the first available Staff handler in the section. Behaviour mirrors Section derived types such as SalesRoom but is intended for assistance.

void HelpDesk::notify ( Person *  person)
virtual

Notify all relevant persons of an event generated by person.

Broadcasts the sender's message to all participants, then invokes the admin's request handler (if set). If no admin is set, the first Staff found in the section will be asked to handle the request.

Parameters
personThe person that triggered the notification.
